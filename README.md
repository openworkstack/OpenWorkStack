# OpenWorkStack (OWS)
Ziel dieses Projekts ist es ein einfacher und übersichtlicher Startpunkt für Unternehmen zu sein, welche eine souveräne digitale Infrastruktur aufbauen möchten.  
Vorangig sollen OpenSource Projekte verlinkt werden mit zusätzlichen Informationen zur Installation und Integration mit anderen in OWS referenzieren Projekten.
